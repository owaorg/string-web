Документация

Полное руководство по использованию STRING

Авторизация

Обязательно: Для работы программы требуется авторизация.
Используйте ваш никнейм для входа в систему.

string --auth ваш_никнейм

Сохраняет ваш никнейм для авторизации в системе. После регистрации перезапустите программу.

string --version

Показывает текущую версию STRING.

Настройка ПО

Необязательно — процедура нужна только для вашего удобства.
Можно легко вернуться к стандартным настройкам.

Интерфейс и внешний вид

string --save on/off

Включает или отключает сохранение отчёта после поиска. Отчёт сохраняется в текущей директории в виде TXT-файла с JSON-содержимым.

string --remove

Сбрасывает все настройки цветов и градиента к заводским значениям.

Градиент и цвета

string --g1 60,60,60 --g2 200,200,200

g1 — начальный цвет градиента в формате RGB (три числа через запятую).
g2 — конечный цвет градиента. Градиент применяется к заголовкам логов.

string --c1 #808080 --c2 #FFFFFF

c1 — первый HEX-цвет для интерфейса.
c2 — второй HEX-цвет. Используется для статического окрашивания элементов.

OSINT Анализ

Бесплатные инструменты для сбора открытой информации.
Используют публичные API и базы данных.

Анализ номера телефона

string --phone +79000000000

Определяет оператора, регион, тип номера (мобильный/городской), местоположение. Использует библиотеку phonenumbers и API voxlink.

WHOIS домена

string --whois example.com

Анализирует домен: DNS-записи (A, NS, MX, TXT, SOA), репутация через VirusTotal, информация о регистрации.

Анализ IP-адреса

string --ip 8.8.8.8

Поиск информации об IP-адресе через базу DepSearch.

Поиск никнейма

string --nickname username

Проверяет наличие никнейма на популярных сайтах: GitHub, VK, Pinterest, TikTok, Reddit, Habr, GitLab и пользовательских сайтах.

string --add-website https://example.com/user/

Добавляет сайт в список для поиска никнеймов. Сайт должен содержать место для подстановки никнейма в конце URL.

Декодирование координат

string --lat 55.7558 --lon 37.6173

Преобразует географические координаты в адрес. Использует Nominatim OpenStreetMap.

Проверка прокси

string --public-proxy 164.90.151.28:3128

Проверяет работоспособность публичного HTTP-прокси.

string --private-proxy proxy:port:login:password

Проверяет приватный прокси с авторизацией. Формат: адрес:порт:логин:пароль.

string --proxy-check C:/path/to/proxy.txt

Проверяет список прокси из файла.

Поиск по базам (DepSearch)

Важно: Поиск происходит по платному источнику DepSearch.
Требуется активная подписка и авторизация.

Поиск по номеру телефона

string --api-phone +79000000000 --api depsearch

Поиск по номеру телефона в базе DepSearch. Требуется параметр --api depsearch.

Поиск по ФИО

string --api-people Иванов Иван Иванович

Поиск по ФИО в базе DepSearch.

Поиск по email

string --api-mail user@example.com --api depsearch

Поиск по почтовому адресу в базе DepSearch. Требуется параметр --api depsearch.

Поиск по VIN-номеру

string --api-vin 1FAFP53U7YG164887

Поиск по VIN-номеру автомобиля. Возвращает историю авто.

Поиск по паролю

string --api-pass password123

Поиск утечек пароля в базах данных.

Поиск по СНИЛС

string --api-snils 541-251-383 52

Поиск по номеру СНИЛС.

Поиск по ИНН

string --api-inn 37507352790

Поиск по ИНН (индивидуальный номер налогоплательщика).

Поиск по профилю ВКонтакте

string --api-vk https://vk.com/username

Поиск информации по ссылке на профиль VK.

Поиск по IP-адресу

string --api-ip 8.8.8.8

Поиск по IP-адресу в базе DepSearch.

Дополнительные инструменты

Обфускация Python-файлов

string --obf script.py

Создаёт обфусцированную копию файла script_string.py. Код кодируется в base64 и упаковывается в exec-обёртку.

Справка

string --shelp

Выводит в консоль краткую справку по всем доступным командам.
